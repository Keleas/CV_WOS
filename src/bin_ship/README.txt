В данном файле находятся:

1) task_img - изображения для теста решений участника
test_image_01, test_image_06, test_image_07 необходимо выложить для скачивания в качестве тренировочных данных
Тут лежит папка arrays c txt файлами массивов изображений 

2) tools/check_ans.py - программа для проверки решения участника, которая выводит на печать True в случае верного решения и False в противном случае, принимает вещественное число - решение участника.

3) reference_solution.py - программа для решения поставленной задачи

4) task.md, task.tex - описание задания

5) tests - входные данные и решения к ним, указать 6% для отклонения от точного решения
